# Mastra AI Summarizer Tutorial

This repository contains a multi-part tutorial that walks through how to build an **AI Summarizer Agent** using **Mastra**. It covers everything from agent setup to UI integration.

---

##  Parts Overview

### **Part 1 — Building the AI Agent**
Learn how to:
- Create a Mastra agent
- Connect tools for summarizing URLs and text
- Understand how data flows through the agent
- Scrape webpage content with **Cheerio**
- Make API calls with **Axios**

[Read Part 1 details](https://github.com/Timmydee/mastra-ai-summarizer-tutorial/tree/main/ai-summarizer-agent)

---

### **Part 2 — Building the Frontend UI**
Learn how to:
- Create a web interface for your AI summarizer
- Handle user input (URLs or text)
- Display formatted summaries from the agent

[Read Part 2 details](https://github.com/Timmydee/mastra-ai-summarizer-tutorial/blob/main/ai-summarizer-agent-ui/)

---

## Tech Stack
- **Mastra** — for building AI agents and tools  
- **Next.js** — frontend framework  
- **TypeScript** — type safety  
- **Tailwind CSS** — UI styling
- **Axios** - HTTP requests to APIs
- **Cheerio** - HTML parsing and scraping for URL summaries

---

##  Getting Started
Clone the repo and navigate into a part:

```bash
git clone https://github.com/Timmydee/mastra-ai-summarizer-tutorial.git
cd mastra-ai-summarizer-tutorial/ai-summarizer-agent
npm install
npm run dev
```
